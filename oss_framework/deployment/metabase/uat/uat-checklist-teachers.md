## Test Scenario 3: Analyze Attendance Impact (15 min)

### Objective
Understand relationship between class attendance and student success.

### Steps

1. [ ] **Review "Attendance Impact on Pass Rate" Scatter Plot**
   - Each dot represents one of your class periods
   - X-axis: Average student attendance in class
   - Y-axis: Pass rate (% with C or better)

2. [ ] **Find Your Highest-Performing Class**
   - Locate the dot for your best class (from Scenario 2)
   - Attendance rate for that class: __________%
   - Pass rate: __________%
   - Is this dot **above** the trend line? [ ] Yes  [ ] No

3. [ ] **Find Your Most Challenging Class**
   - Locate the dot for your struggling class
   - Attendance rate for that class: __________%
   - Pass rate: __________%
   - Is this dot **below** the trend line? [ ] Yes  [ ] No

4. [ ] **Interpret the Pattern**
   - If dot is **on trend line**: Pass rate is expected given attendance
   - If dot is **above trend line**: You're getting better results than expected (strong instruction)
   - If dot is **below trend line**: Pass rate is low even with decent attendance (may indicate instructional challenge)
   
   - For your challenging class, is it:
     - [ ] Attendance-driven (low attendance, low pass rate, on trend line)
     - [ ] Instruction-driven (decent attendance, low pass rate, below trend line)
     - [ ] Both (very low attendance, very low pass rate)

5. [ ] **Action Planning**
   - If attendance-driven → Focus on engagement strategies, parent contact
   - If instruction-driven → May need differentiation, pacing adjustments, or peer observation
   - Does this insight help you? [ ] Yes  [ ] No
   - What action will you take? _________________________________________

### Success Criteria
✅ Can distinguish attendance vs. instructional factors  
✅ Scatter plot is intuitive and actionable  
✅ Provides diagnostic insight for improvement

### Objective
Review your grading patterns compared to school norms.

### Steps

2. [ ] **Analyze Your Distribution**

     - A: ______%
     - B: ______%
     - C: ______%
     - D: ______%
     - F: ______%

3. [ ] **Compare to School Average**
   - Is your distribution similar to school average? [ ] Yes  [ ] No
   - If No, which grade category is most different? __________

4. [ ] **Reflection**
   - If you have more As/Bs than average:
